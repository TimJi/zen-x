# Chrome Web Store Screenshots Guide

## Required Screenshots

Chrome Web Store requires at least **1 screenshot**. Recommended: 2-3.

### Dimensions
- **1280 x 800** px (preferred) or **640 x 400** px
- PNG or JPEG format

### Recommended Screenshots

1. **Before** — A normal X.com article page with all distractions visible (sidebar, comments, navigation, ads)
2. **After** — The same article page after clicking ZenX, showing the clean zen reading mode

### How to Capture

1. Open an X.com article page in Chrome
2. Set browser window to 1280px wide (use DevTools → device toolbar or resize window)
3. Take a screenshot of the full page (**Before**)
4. Click the ZenX extension icon
5. Take another screenshot (**After**)

Tip: Use Chrome's built-in screenshot (DevTools → Ctrl+Shift+P → "Capture screenshot") for exact dimensions.

## Promotional Image (Optional but Recommended)

- **440 x 280** px
- Displayed on the Chrome Web Store listing card
- Should be a branded graphic showing the extension concept (e.g., ZenX logo + before/after split view)

## Store Icon

Already provided in the extension package:
- 128 x 128 px (`icons/icon128.png`) — used as the store icon
